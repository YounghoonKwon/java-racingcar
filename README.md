# java-racingcar
자동차 경주 게임 미션 저장소

## 기능 요구사항
  - [ ] 경주할 자동차 이름을 입력받는다. (쉼표(,)로 구분)
    - [ ] 예외 : 이름 5자 이하, 빈칸 trim
  - [ ] 시도할 횟수를 입력받는다.
  - [ ] 0부터 9사이의 랜덤 값으로 자동차를 전진시킨다.
    - [ ] 전진 조건 : 랜덤 값 4 이상
    - [ ] 멈춤 조건 : 랜덤 값 3 이하
  - [ ] 실행 결과(전진 횟수)를 출력한다.
  - [ ] 최종 우승자를 출력한다.
    - [ ] 공동 우승자가 가능하다.
